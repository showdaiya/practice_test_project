# ユーザビリティ・アクセシビリティ

## 1. スロット予約UI
- 空きが無い場合は最短の代替スロットを**即時提案**
- 保留中スロットのTTLをカウントダウン表示

## 2. クーポン入力
- 入力時に**即時検証**（D-01に準拠）、理由コードを短文で提示
- 複数クーポン不可ならUIで明示し、重複入力を無効化

## 3. A11y
- WCAG 2.1 AA準拠、キーボード操作可能、フォーカス可視
- ARIA属性で状態・エラーを告知、スクリーンリーダー向けテキスト
